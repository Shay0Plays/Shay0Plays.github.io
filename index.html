Theme

I use a custom client theme on Discord to personalize my experience to how I want it to be and make the interface more visually appealing, this includes removing unnecessary clutter and changing the colour. It uses the colours of Catppuccin Mocha, with the mauve accent (Just like the website) If you're interested in checking it out or using it yourself, you can use the css below or just enter the url as an online theme.

@import url('https://raw.githubusercontent.com/Shay0Plays/Shay0Plays.github.io/refs/heads/main/colour.theme.css')

If you do not want to use this colour scheme then use the one below, it will also make it so that you can use other themes that change the colour of the client on top of this

@import url('https://raw.githubusercontent.com/Shay0Plays/Shay0Plays.github.io/refs/heads/main/nocolour.theme.css')

N.B. This also modifies the SpotifyControls plugin from Vencord
